Oxbury PLC test

You can run these scripts

### `npm install`

### `npm start`

to start the server : listening on prot 8000
e.g http://localhost:8000/api/beers` to get list of beers
or
http://localhost:8000/api/beers/?bname=buzz` to get beer called buzz

### `npm run start-ui`

to start the UI

http://localhost:3000
to access the ui
